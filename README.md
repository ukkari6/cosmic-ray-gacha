# 🌌 宇宙線ガチャ生活 (Cosmic Ray Gacha Life)

「宇宙線は本当に地上のコンピュータのメモリ（RAM）を反転させるのか？」を、飽きるまで毎日と検証し続けるプロジェクトです。
各種サイトで配信中の動画『宇宙線ガチャ生活』のリポジトリです。

## プロジェクトの目的と透明性
ネットミーム化しつつある「バグ＝宇宙線のせい」という言説に対し、本当にそんなことが起きるのかとりあえずやってみることを目的としています。

検証プログラムのミスやデバッグ不足による誤作動を排除し、宇宙線によるメモリのビットの反転を捉えるため、使用しているソースコードおよび検証スクリプトをここに公開します。

---

## 実験環境とハードウェア構成

### 開発・検証環境
- **OS**: Xubuntu 24.04.4 LTS

本プロジェクトでは、Raspberry Pi Picoボードを空に何も遮るものが無い場所に一晩中設置しています。

### 環境対策
宇宙線以外の原因でメモリが化けるのを防ぐため、以下の対策を施しています。
- **熱対策**: 直射日光を避け、夜間〜早朝の涼しい時間帯のみ運用。
- **電源対策**: モバイルバッテリーを使用し、マイコンの電源ピンの近くに電解コンデンサを設置。
- **虫対策**:虫による文字通りバグが起きないように、マイコンとモバイルバッテリーはジップロックに入れます。

---

## 仕組みとコードの構造

### ファームウェア側（C言語）
1. 起動時に、RAMの特定領域を `0x55,0xAA`で埋め尽くします。
2. その後、メインループは `while(1);` で完全に待機状態（静的保持状態）に入り、宇宙線がメモリにヒットするのを待ちます。
3. 翌朝、物理スイッチを押すことで、保持していたメモリの内容をUART経由で生データとしてPCへ送信します。

### PC側（Python）
マイコンから送られてきたデータを1バイトずつチェックし、`0x55,0xAA` 以外のデータが存在しないかスキャンします。

## 使い方 / 検証の再現方法

### 1. ラズパイピコの準備
各 `firmware/` ディレクトリのソースコードを書き込みしてください。

### 2. PC側スクリプトの実行
依存ライブラリをインストール後、マイコンをUSB（シリアル）で接続してスクリプトを実行します。
